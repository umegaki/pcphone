# PcPhone
* PC Phone is a telephone assist device for physical disabilities. 
* PCホンは、肢体不自由のために固定電話機の操作ができない人が、PCから固定電話を操作できるようにするデバイスです。
* 原則として、オープンハードウェアとして公開していますが、ライセンス条件は、まだ書きかけです。変更される可能性があります。

# 構成

次の3つで構成されます。

1. 固定電話機（要改造）
2. PcPhone Arduino コントローラ
3. PcPhone Windows アプリケーション

# 注意喚起

電話機とコントローラの接続には電話機の改造が必要になります。このような改造は、技術基準適合認定（*技適*）に違反することになります。特に、電波を発する機器の場合には、その運用はより厳しくなります。したがって、通信回線にかかわる部分を改造しない、無線を搭載した固定電話機を使用しない、直接電話回線網に接続せず ONUやルーターを介して接続する、という点に注意を払いました。しかし、改造それ自体は違法とされる可能性のある行為です。今回は、障害者が回線を使えなければ利用できないサービスがあることを前提にして、電気通信事業法の趣旨を尊重しつつ、障害者の情報アクセスの権利を確保するにはやむをえないと判断しました。本機のユーザは、サービスを利用できない旨を当該運営会社に連絡し対応策を求めました。当該企業の担当者はそのユーザを訪問し現状を把握しましたが、今のところ対応策はないという結論でした。その後、ユーザから作者に相談を持ち掛けられました。

サービスを提供する運営会社は、障害者がサービスを利用できない状態であることに対して苦情を申し立てているのですから、合理的配慮に基づいた対策をすべきであり、それが本来行われるべき対策です。本機の開発はそのようなサポート不在に対するやむを得ない解決策であったことも付け加えておきます。

また、作者は、アナログ第2種工事担当者資格（旧電話工事資格）を持っており、その責任において、回線網に影響を与えないよう十分に配慮して設計しました。

作者は、このように不法性を問われるリスクを冒すことを問題なしとは考えません。障害者に対する合理的配慮の欠如によって生じる問題であり、このようなデバイスは本来必要なものではないし、またその制作を推進する立場にもありません。ここで公開する情報を利用して何らかの機器開発を行うときには、ここで述べた事項について自身の責任において実施してください。

#  PcPhone Arduino コントローラ

PCからのコマンドを受け付けて、電話機のボタンを押すための制御部です。

* Arduino nano R3 を用いました。
* 電話機の0～9、#*、ハンズフリ－の各ボタンの配線を確認して、それらをリレーを用いてオンすることによって、電話をかけることができる構造です。
* PCとの通信は、Arduino の USB と統合された仮想シリアルポートを通じて行います。
* リレーの個数を節約するために、電話機のボタンをスキャンする信号 COL、ROW を個別にオンします。この仕様は、電話機のボタンのスキャン方式に依存します。
* 半導体スイッチではなくメカニカルリレーを使用した理由は、できるだけ電話機本体と疎結合にすること、サージなどの影響を受けても発火・感電事故などが起きにくくするためです。
* 電話機との接続には、14ピンのフラットケーブルを使用していますが、ケーブルやボックスヘッダーが秋月電子通商で安価に購入できることが理由です。
* タカチ電機工業 LC型ハンドタイププラスチックケース(オフホワイト) LC135H-N-W に収まるようにJLCPCBでプリント基板を製造しました。

# 固定電話機

* シンプルシニアホン ブラック_TEL-2991SO-K 05-2991 （オーム電機）を使用しました。 [https://www.ohm-electric.co.jp/product/c17/c1704/37945/]
* 国産で、技適があり、アダプタ不要で回線だけで動作する、ハンズフリー機能のあるものを選択しました。
* この電話機の制御には、Silan Semiconductor 社の SC91415 が使用されています。
* ここにデータシートがあります。[https://www.digchip.com/datasheets/parts/datasheet/433/SC91415-pdf.php]
