# PcPhone
* PC Phone is a telephone assist device for physical disabilities 
* PCホンは、肢体不自由のために固定電話機の操作ができない人が、PCから固定電話を操作できるようにするデバイスです。

# 構成

次の3つで構成されます。

1. 固定電話機（要改造）
2. PcPhone arduino コントローラ
3. PcPhone Windows アプリケーション

# 注意喚起

電話機とコントローラの接続には電話機の改造が必要になります。このような改造は、技術基準適合認定（技適）に違反することになります。特に、電波を発する機器の場合には、その運用はより厳しくなります。したがって、通信回線にかかわる部分を改造しない、無線を搭載した固定電話機を使用しない、直接電話回線網に接続せず ONUやルーターを介して接続する、という点に注意を払いました。しかし、改造それ自体は違法です。今回は、障害者が回線を使えなければ利用できないサービスがあることを前提にして、電気通信事業法の趣旨を尊重しつつ、障害者の情報アクセスの権利を確保するにはやむをえないと判断しました。本機のユーザは、サービスを利用できない旨を当該運営会社に連絡し対応策を求めました。当該企業の担当者はそのユーザを訪問し現状を把握しましたが、今のところ対応策はないという結論でした。その後、ユーザから作者に相談を持ち掛けられました。

サービスを提供する運営会社は、障害者がサービスを利用できない状態であることに対して苦情を申し立てているのですから、合理的配慮に基づいた対策をすべきであり、それが本来行われるべき対策です。本機の開発はそのようなサポート不在に対するやむを得ない解決策であったことも付け加えておきます。

また、作者は、アナログ第2種工事担当者資格（旧電話工事資格）を持っており、その責任において、回線網に影響を与えないよう十分に配慮して設計しました。

作者は、このように不法性を問われるリスクを冒すことを問題ないとは考えません。この問題は、障害者に対する合理的配慮の欠如によって生じる問題であり、このようなデバイスは本来必要なものではないし、またその制作を推進する立場にもありません。ここで公開する情報を利用して何らかの機器開発を行うときには、ここで述べた事項について自身の責任において実施するようにしてください。
