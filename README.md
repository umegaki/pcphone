# PcPhone
* PC Phone is a telephone assist device for physical disabilities 
* PCホンは、肢体不自由があり固定電話機の番号操作ができない人が、PCから固定電話を操作できるようにするデバイスです。

# 構成

次の3つで構成されます。

1. 固定電話機（要改造）
2. PcPhone arduino コントローラ
3. PcPhone Windows アプリケーション

# 注意喚起

電話機とコントローラの接続には電話機の改造が必要になります。このような改造は、技術基準適合認定（技適）に違反することになります。特に、電波を発する機器の場合には、その運用はより厳しくなります。したがって、通信回線にかかわる部分を改造しない、無線を搭載した固定電話機を使用しない、直接電話回線網に接続せず ONUやルーターを介して接続する、という点に注意を払いました。しかし、改造それ自体は違法です。今回は、障害者が回線を使えなければ利用できないサービスがあることを前提にして、電気通信事業法の趣旨を尊重しつつ、障害者の情報アクセスの権利を確保するにはやむをえないと判断しました。

また、作者は、アナログ第2種工事担当者資格を持っており、その責任において、回線網に対する影響を与えないよう十分に配慮して設計しました。
